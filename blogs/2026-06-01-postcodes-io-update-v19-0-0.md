Postcodes.io v19.0.0 adds enhanced regional data and performance improvements, including updated Welsh Parliament constituency data and expanded Scottish postcode coverage with over 50 data fields. The release simplifies the database structure into three master tables and optimizes query performance through batching and PostgreSQL tuning.
